Optiunile meniului

1. Sa se gaseasca numarul, valoarea, data chitantelor pentru un apartament in ordinea datei

2. Sa se gaseasca apartamentele cu suprafata mai mare decat o suprafata data

3. Sa se gaseasca nume_proprietar, adresa_apartament1, adresa_apartament2 pentru apartamente ce apartin aceluasi propretar (o combinatie de doua apartamente este unica in rezultat )

4. Sa se gaseasca nume_proprietar, adresa, an luna, valoare pentru apartamentele de pe strada introdusa

5. Sa se gaseasca adresa apartamentelor si luna cu cea mai mica valoare pentru anul introdus

6. Sa se gaseasca numele proprietarilor ce detin mai multe apartamente

7. Sa se gaseasca valoare maxima pentru fiecare apartament

8. Sa se gaseasca numele restantierilor (suma chitantelor < suma cheltuielilor )
